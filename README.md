DealerHub-"ডিলার পয়েন্টের প্রতিটি হিসাব, এখন আপনার হাতের মুঠোয়।" 
এটি সিগারেট ও এফএমসিজি ডিস্ট্রিবিউশনের জন্য তৈরি একটি কমপ্লিট ওয়েব-ভিত্তিক ম্যানেজমেন্ট সল্যুশন। দৈনিক বিক্রয়, সিএসআর পারফরম্যান্স, স্টক ট্র্যাকিং, ইনসেন্টিভ/ক্যাশব্যাক হিসাব, খরচ নিয়ন্ত্রণ এবং অটোমেটেড রিপোর্টিং – সবকিছু এক ড্যাশবোর্ডে রিয়েল-টাইম ম্যানেজ করুন।
